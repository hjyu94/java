Tree: 계층, 그룹. 노드가 1개 이상의 자식을 가지면 트리라고 한다.
leaf: 자식이 없는 노드

# Binary Tree (이진 트리): child가 최대 2개인 트리
# Binary Search Tree (이진 검색 트리): 현재 노드의 왼쪽 자식 노드는 현재 노드보다 작아야 하고 오른쪽은 반대.
# Complete Binary Tree (완전 이진 트리): 마지막 레벨을 제외한 모든 서브트리의 레벨이 같아야 하고 마지막 레벨은 왼쪽부터 채워져 있으면 된다.
# Full Binary Tree: 모든 노드의 자식의 개수가 0개 혹은 2개인 트리
# Perfect Binary Tree: 모든 자식의 개수가 2개인 트리 (전체 노드 갯수가 2^(level) - 1개)
