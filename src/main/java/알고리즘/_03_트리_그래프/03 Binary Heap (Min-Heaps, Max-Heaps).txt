# Heap?
힙이란 최대값이나 최소값을 빨리 하기 위해 고안된 완전 이진트리를 기본으로 한 자료구조

# Min-Heaps?
작은 값을 항상 위의 노드에 존재하게 해서, 루트에는 가장 작은 값이 오도록 한다.

                1
        2               3
    4       5

# Max-Heaps?
큰 값을 항상 위의 노드에 존재하게 해서, 루트에는 가장 큰 값이 오도록 한다.

                5
        4               3
    1       2

# 최소 힙에 노드 삽입하기 ( O(logn) )
    1. 완전 트리의 맨 끝에 노드를 삽입한다.
       (왼쪽부터 채워나가야 한다)
    2. 자신의 부모노드와 비교해서 자기가 값이 작으면 부모 노드와 자리를 바꾼다.
    3. 이를 반복 (노드가 루트에 도착할 때까지 반복)

# 최소 힙에서 노드 꺼내오기 ( O(logn)) -> 가야할 길이 반씩 뚝뚝 떨어지니까
    1. 가장 작은 값은 루트에서 꺼낸다.
    2. 완전 이진트리의 맨 마지막 노드를 가져와서 루트에 채운다.
    3. 정렬
        3-1. 자신의 자식 노드 중 작은 놈과 자리를 바꾼다.
             (왼쪽, 오른쪽 둘다 부모보다 작다면 더 작은 놈과 자리를 바꿈)
        3-2. 반복
        3-3. 자식이 둘 다 자기보다 크거나 leaf 에 도착하면 스땁
