# 그래프의 탐색 방법
[1] Depth-First Search (DFS, 깊이 우선 탐색)
    - inorder
    - preorder
    - postorder
    - Stack 으로 구현

[2] Breadth-First Search (BFS, 너비 우선 탐색)
    - 레벨 단위로 검색하는 것.
    - Queue 로 구현


# DFS 구현 (스택, 꺼담출)
    1. 시작 노드를 스택에 담는다.
    2. 스택에서 노드를 하나 꺼내고
    3. 꺼낸 노드의 인접한 노드가 있다면 스택에 담고 마킹한다.
      (단 예전에 스택에 들어간적이 있던 노드라면 담지 않는다)
    4. 꺼낸 노드를 출력한다
    5. 2~4 반복
    6. 스택이 모두 비었다면 순회가 완료 된 것.

# BFS 구현 (큐, 꺼담출)
    1. 시작 노드를 큐에 넣는다.
    2. 큐에서 노드를 하나 꺼내고
    3. 꺼낸 노드의 인접한 노드가 있다면 큐에 담고 마킹한다.
      (단 예전에 큐에 들어간적이 있던 노드라면 담지 않는다)
    4. 꺼낸 노드를 출력한다.
    5. 2~4 반복
    6. 큐가 모두 비었다면 순회가 완료 된 것.

- 그래프는 트리가 아니니까 시작노드가 정해져 있지 않다.

# DFS Recursion
    1. 노드에 방문하면 데이터를 출력
    2. 인접한 노드 들을 순서대로 재귀호출 한다.
    // 반환하기 전에 인접한 노드들을 먼저 호출하기 때문에 재귀호출로 깊이 우선검색이 가능하다.
