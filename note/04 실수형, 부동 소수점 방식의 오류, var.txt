@ 실수형/부동 소수점 방식의 오류/var

# 실수형
    - 부동 소수점 방식 사용
    1.0 x 10^-1
    1.0: 가수
    10: 밑수, -1: 지수

#  부동 소수점 방식의 오류
    : 정확히 0을 표시할 수 없다.
    : 대신 다양한 범위의 소수 표현 가능.

    3.14: 리터럴: 기본적으로 double로 저장.
    (정수형은 기본적으로 리터럴을 int에 저장)
    double dNum = 3.14; (O)
    float fNum = 3.14; (X)
    float 는 4바이트라서 8바이트인 리터럴을 저장할 수 없음. 비좁아서 안됨. 그래서 3.14뒤에 f를 붙여야 한다.
    float fNum = 3.14f; (O)

# var
    - 지역변수에 한해서 자료형의 타입을 명시하지 않아도 된다.
    - 변수에 대입되는 값을 보고 컴파일러가 추론.

    var iNum = 10;
    var isTrasparent = false;
